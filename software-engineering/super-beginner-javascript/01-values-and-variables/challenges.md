# Challenges — Values and Variables

> Practice storing and changing named values with five short challenges.

These challenges are intentionally smaller than the main homework. They are meant to help a beginner get more repetitions without feeling overwhelmed.

## How to use these

1. Pick one challenge at a time.
2. Predict what your code should do before you run it.
3. Keep the script small enough to explain line by line.
4. If you get stuck, go back to the demo and change only one thing at a time.

## The five challenges

1. Create three variables for a player name, score, and level, then print all three.
2. Change the score variable twice and print the result after each update.
3. Create one `const` and one `let`, then explain why they are different.
4. Store a favorite food in a variable, then change it to a different food and print both versions one after the other.
5. Write a script that stores your name in one variable and a greeting in another, then combines them in one `console.log`.

## Done when

- you completed at least three of the five without copying blindly
- you can explain your solution in plain language
- you can change one input or value and still predict the new output
