# Challenges — Strings, Numbers, and Booleans

> Practice recognizing and using the most common JavaScript value types with five short challenges.

These challenges are intentionally smaller than the main homework. They are meant to help a beginner get more repetitions without feeling overwhelmed.

## How to use these

1. Pick one challenge at a time.
2. Predict what your code should do before you run it.
3. Keep the script small enough to explain line by line.
4. If you get stuck, go back to the demo and change only one thing at a time.

## The five challenges

1. Create one string, one number, and one boolean, then print them with labels.
2. Add two numbers together and print the result.
3. Create a boolean called `isHungry` and print a sentence explaining whether it is true or false.
4. Turn a name and age into one sentence using a string and a number.
5. Make a small script that tracks whether a student passed a quiz using a boolean and a score.

## Done when

- you completed at least three of the five without copying blindly
- you can explain your solution in plain language
- you can change one input or value and still predict the new output
