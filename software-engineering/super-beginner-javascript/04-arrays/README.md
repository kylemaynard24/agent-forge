# Arrays

**Track:** Super beginner JavaScript

## What this topic is about

Store ordered lists of values and work through them step by step.

## Why it matters

If someone is absolutely new to programming, arrays is one of the first places where code stops looking like random punctuation and starts looking like a set of ideas. This topic helps a learner see that programs are not magic; they are just instructions acting on values in predictable ways.

## When to use it

- you want to write code for keeping a shopping list or a list of class names
- you need one small building block before moving to larger programs
- you want to explain not just what the syntax is, but why it exists

## The core idea

Start simple:

1. name the thing you are working with
2. inspect what value it currently has
3. change it or use it deliberately
4. explain the result in plain language

That habit is more important than memorizing syntax.

## Common beginner mistakes

- copying code without predicting what it will print
- changing several things at once and losing track of the cause
- memorizing syntax before understanding the underlying idea

## Tiny example

Imagine keeping a shopping list or a list of class names. Store ordered lists of values and work through them step by step. Once the learner can explain the example out loud, the code starts to become intuitive instead of scary.
