# Project — Loops

## Goal

Build a beginner-friendly mini project that uses **Loops** in a way a new learner can understand and explain.

## Suggested idea

Start from printing every task in a todo list and expand it slightly so it feels like a tiny real program instead of a single isolated line.

## Deliverables

1. A runnable `demo.js`-style script
2. A short note explaining the program in everyday language
3. One version of the code before improvement
4. One version after you make it clearer or more useful

## Constraints

- the project must stay small enough to explain in 5-10 minutes
- the learner should be able to point to each line and say what it does
- avoid advanced syntax that hides the basic idea

## Done when

- a beginner could read the script and follow it
- the project uses the topic clearly rather than accidentally
- you could teach it live without needing to improvise the explanation
