# Challenges — Scope and State

> Practice understanding where variables live and how program state changes with five short challenges.

These challenges are intentionally smaller than the main homework. They are meant to help a beginner get more repetitions without feeling overwhelmed.

## How to use these

1. Pick one challenge at a time.
2. Predict what your code should do before you run it.
3. Keep the script small enough to explain line by line.
4. If you get stuck, go back to the demo and change only one thing at a time.

## The five challenges

1. Create a variable outside a function and update it inside the function, then print it.
2. Write one function that changes a score variable and another that prints it.
3. Create two variables with similar names in different places and explain which one changes.
4. Make a simple step counter that increases every time a function runs.
5. Write down what value you expect a variable to have before and after calling a function.

## Done when

- you completed at least three of the five without copying blindly
- you can explain your solution in plain language
- you can change one input or value and still predict the new output
