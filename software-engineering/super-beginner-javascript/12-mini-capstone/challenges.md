# Challenges — Mini Capstone

> Practice combining beginner JavaScript concepts into one small explainable program with five short challenges.

These challenges are intentionally smaller than the main homework. They are meant to help a beginner get more repetitions without feeling overwhelmed.

## How to use these

1. Pick one challenge at a time.
2. Predict what your code should do before you run it.
3. Keep the script small enough to explain line by line.
4. If you get stuck, go back to the demo and change only one thing at a time.

## The five challenges

1. Build a tiny todo list script using an array of objects.
2. Build a score tracker that uses variables, conditionals, and functions together.
3. Create a simple student record program that stores a few students and prints a summary.
4. Write a mini shop script that adds item totals and prints a final message.
5. Choose one of your earlier beginner topics and combine at least three concepts into one small app.

## Done when

- you completed at least three of the five without copying blindly
- you can explain your solution in plain language
- you can change one input or value and still predict the new output
