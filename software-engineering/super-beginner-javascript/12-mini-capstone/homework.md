# Homework — Mini Capstone

> Combine the beginner topics into one small program you can explain confidently.

## Exercise

Create a small script related to building a console-based student tracker, task app, or mini shop.

**Build:**
- one clear example using the topic correctly
- one small variation where you change the input and predict the output first
- a short explanation in plain English of what the code is doing

**Constraints:**
- keep the script small enough to explain line by line
- do not copy code without changing at least one value yourself
- write down what you expect before you run it

## Reflection

- What part of Mini Capstone felt the most natural?
- What part still feels confusing or easy to mix up?
- Could you teach this exact example to a friend without reading the file?

## Done when

- you can run the script successfully
- you can predict the output before running it
- you can explain the code in plain language

## Clean Code Lens

**Principle in focus:** The Whole Program as a Communication Device

The capstone is where everything you have learned converges — and where naming debt accumulates fastest. When multiple concepts interact (variables feeding functions feeding objects feeding loops), a single unclear name can force a reader to trace the entire program to understand one line. Clean code at this scale means treating every identifier as a word in a conversation with a new teammate.

**Exercise:** When your capstone runs correctly, go back through every variable, function, and object property and ask: "Would a new teammate understand this name without asking me?" Rename anything that requires prior knowledge of your thought process to understand. Then read the whole program top to bottom, out loud, as if narrating it. If you stumble, that is where a name needs improving.

**Reflection:** Which part of the capstone would be hardest to hand off to someone else right now — and is that difficulty caused by the problem being genuinely complex, or by names and structure that could be made clearer?
