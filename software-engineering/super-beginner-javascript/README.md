# super-beginner-javascript

A beginner-first JavaScript path for people who are completely new to programming and for friends who want to learn how code works from the ground up.

This section is intentionally slower, simpler, and more explicit than the rest of the repo. It starts from:

1. values
2. variables
3. conditionals
4. loops
5. functions
6. objects
7. classes

The goal is not to rush. The goal is to build the habit of understanding what code is doing, line by line.

## How to teach from this section

Each topic folder contains:

1. **`README.md`** — the idea in plain language
2. **`demo.js`** — a tiny runnable example
3. **`homework.md`** — a beginner-friendly exercise
4. **`challenges.md`** — five extra practice prompts for repetition
5. **`project.md`** — a slightly larger teaching project

The best teaching loop is:

1. read the README together
2. predict what the demo will print
3. run the demo
4. change one value
5. predict again
6. do the homework
7. build the mini project

## Teaching rule of thumb

If a learner can run the code but cannot explain it, slow down.

If a learner can explain it in everyday language, they are making real progress.

## Question-driven orientation

### "When should I spend time in this section?"

**Answer:** Spend time here when you want to improve how to help a learner move from zero to real confidence.

**Explanation:** A section guide is most useful when it helps you choose the right learning path on purpose. The point is not to read everything at once. The point is to understand what kinds of problems this section trains you to notice and solve.

### "How should I study this without turning it into passive reading?"

**Answer:** Read the concept, predict the demo, run it, change something small, and explain the result out loud.

**Explanation:** This repo works best when each README turns into a decision habit. If you can explain when to use the idea, when not to use it, and what trade-off it introduces, the topic is starting to become real knowledge instead of reference material.
